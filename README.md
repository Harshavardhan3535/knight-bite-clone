# Knight Bite Clone

A static clone of the Knight Bite website built using HTML and CSS as part of frontend development practice.

## Features

- Responsive webpage layout
- Navigation menu
- Hero section
- Multiple content sections
- Custom styling with CSS

## Technologies Used

- HTML5
- CSS3

## Project Structure

```text
knight-bite-clone/
│
├── index.html
├── style.css
├── Images/
└── README.md
```

## Purpose

This project was created to improve frontend development skills by recreating an existing website layout using HTML and CSS.
